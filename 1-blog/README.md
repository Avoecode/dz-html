Блог о веб-разработке – «Теория цвета»

Вёрстка (HTML/CSS) выполнена по учебному макету – задание на отработку семантики, флексов.

Добавлено мной самостоятльно:

Адаптивность 

- Корректное отображение на всех устройствах (десктоп, планшет, мобильные телефоны)
- Медиазапросы для брейкпоинтов `1200px`, `1024px`, `768px`
- Адаптивная шапка, скрытие навигации на планшетах, мобильная кнопка поиска


- Hover-эффект меню – при наведении цвет меняется , переход плавный


Javascript :

- Плавная прокрутка – при клике на пункты меню страница плавно скроллится к соответствующему разделу (`#html-css` или `#backend`)
- Активный пункт меню – текущий раздел подсвечивается (жирный шрифт, `opacity: 1`)
- Переключение тёмной/светлой темы – кнопка с иконками 🌙 / 🌞 меняет фон страницы (градиент)
- Сохранение темы в localStorage – при перезагрузке страницы выбранная тема восстанавливается, сохраняется для пользователя
