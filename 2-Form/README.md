# Форма регистрации с валидацией

Проект представляет собой форму регистрации с полями:

- **Email**
- **Имя / Фамилия** (два поля в одной строке)
- **Пароль**
- **Чекбокс** (согласие с условиями)

## 📌 Как создавался проект

Изначально это было учебное задание по вёрстке макета из Figma – требовалось сверстать форму только с помощью HTML и CSS.  
Вёрстка выполнена мной по макету, проверена наставником.

## ✨ Что я доработала самостоятельно

- **Адаптив** – форма корректно отображается на планшетах и мобильных устройствах (медиа-запросы, гибкие паддинги).
- **Валидация на чистом JavaScript**:
  - Проверка email и пароля (регулярные выражения).
  - Групповая валидация для полей имени и фамилии – ошибки выводятся под общим блоком, не ломая строку.
- **Сохранение черновика в `localStorage`** – введённые данные не теряются после перезагрузки страницы.
